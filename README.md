# 소개
무선 LAN 및 Bluetooth 연결이 가능한 단일 보드 컴퓨터인 Raspberry Pi 3b와 온습도 센서인 dht22을 연결
인터넷 또는 LAN을 통해 HTTP프로토콜을 사용하여 물건의 데이터를 저장 및 검색 하는 오픈 소스 Internet of Thing (IoT) 응용 프로그램 및 API인 ThingSpeak에 정보를 업로드
